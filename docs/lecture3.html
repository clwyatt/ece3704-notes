Convolution in CT

Recall from ECE 2714 that linearity and time-invariance, plus the sifting property of the delta function leads directly to the convolution integral for CT systems analysis. We consider an input signal x(t)x(t) to be composed of an unaccountably infinite linear combination of weighted delta functions

x(t)=∫−∞∞x(τ)δ(t−τ)dτx(t) = \int\limits_{-\infty}^{\infty} x(\tau)\delta(t-\tau)\; d\tau

By linearity then the output is given by

y(t)=∫−∞∞x(τ)h(t−τ)dτ≡x*h=h*xy(t) = \int\limits_{-\infty}^{\infty} x(\tau)h(t-\tau)\; d\tau \equiv x * h = h * x

where the signal h(t)h(t) is the impulse response of the system.

This approach to analysis works for any CT LTI system, stable or unstable, as long as the convolution integral converges.

We can build up a table of convolutions by evaluating the convolution integral for a set of primitive functions. Some examples

δ(t)*x(t)=∫−∞∞δ(τ)x(t−τ)dτ from definition of convolution=x(t) by the sifting property of delta function\begin{aligned} \delta(t) * x(t) &= \int\limits_{-\infty}^{\infty} \delta(\tau)x(t-\tau)\; d\tau & \text{ from definition of convolution}\\ &= x(t) & \text{ by the sifting property of delta function} \end{aligned}

Example 2: Convolution of two step functions:

u(t)*u(t)={0t<0∫0tdτt≥0={0t<0tt≥0=t⋅u(t)\begin{aligned} u(t) * u(t) &= \left\{\begin{array}{cc} 0 & t < 0\\ \int\limits_{0}^{t} d\tau & t \geq 0 \end{array}\right.\\ &= \left\{\begin{array}{cc} 0 & t < 0\\ t & t \geq 0 \end{array}\right.\\ &= t\cdot u(t) \end{aligned}

Example 3: Convolution of causal exponential eatu(t)e^{a t}\, u(t), a≠0a \neq 0, and unit step:

eatu(t)*u(t)={0t<0∫0teaτdτt≥0={0t<01a(eat−1)t≥0=eat−1a⋅u(t)\begin{aligned} e^{a t}u(t) * u(t) &= \left\{\begin{array}{cc} 0 & t < 0\\ \int\limits_{0}^{t} e^{a \tau}d\tau & t \geq 0 \end{array}\right.\\ &= \left\{\begin{array}{cc} 0 & t < 0\\ \frac{1}{a}\left(e^{at} - 1\right) & t \geq 0 \end{array}\right.\\ &= \frac{e^{at} - 1}{a}\cdot u(t) \end{aligned}

  • commutivity: x1(t)*x2(t)=x2(t)*x1(t)x_1(t) * x_2(t) = x_2(t) * x_1(t)

  • distributivity: x1(t)*[x2(t)+x3(t)]=[x1(t)*x2(t)]+[x1(t)*x3(t)]x_1(t) * [x_2(t) + x_3(t)] = [x_1(t)*x_2(t)] + [x_1(t)*x_3(t)]

  • associativity: x1(t)*[x2(t)*x3(t)]=[x1(t)*x2(t)]*x3(t)x_1(t) * [x_2(t) * x_3(t)] = [x_1(t)*x_2(t)] * x_3(t)

  • time shift: x1(t−τ1)*x2(t−τ2)=[x1*x2](t)|t→τ1+τ2x_1(t-\tau_1)*x_2(t-\tau_2) = \left. [x_1 * x_2](t) \right|_{t\rightarrow \tau_1 + \tau_2}

  • multiplicative scaling, a,b∈ℂa,b\in\mathbb{C}: [ax1(t)]*[bx2(t)]=ab[x1(t)*x2(t)][ax_1(t)]*[bx_2(t)] = ab[x_1(t) * x_2(t)]

DT Convolution

For DT systems the convolution integral of CT system theory becomes an countably infinite sum. Using the sifting property of the DT delta function we can write an input signal as the sum of weighted, time-shifted impulse functions

x[n]=∑m=−∞∞x[m]δ[n−m]x[n] = \sum\limits_{m = -\infty}^{\infty} x[m]\delta[n-m]

By linearity and time-invariance then the output is given by

y[n]=∑m=−∞∞x[m]h[n−m]y[n] = \sum\limits_{m = -\infty}^{\infty} x[m]h[n-m] where h[n]h[n] is the impulse response of the DT system.

As for CT systems, this analysis works for any DT LTI system as long as the sum converges.

We can build up a table of convolution results. Some examples

Example 4: Convolution of delta with any function:

δ[n]*x[n]=∼m=−∞∞δ[m]x[n−m] from definition of convolution=x[n] by the sifting property of delta function\begin{aligned} \delta[n] * x[n] &= \sim\limits_{m = -\infty}^{\infty} \delta[m]x[n-m] & \text{ from definition of convolution}\\ &= x[n] & \text{ by the sifting property of delta function} \end{aligned}

Example 5: Convolution of two step functions:

u[n]*u[n]={0n<0∑m=0n(1)n≥0={0n<0n+1n≥0=n⋅u[n]\begin{aligned} u[n] * u[n] &= \left\{\begin{array}{cc} 0 & n < 0\\ \sum\limits_{m = 0}^{n} (1) & n \geq 0 \end{array}\right.\\ &= \left\{\begin{array}{cc} 0 & n < 0\\ n+1 & n \geq 0 \end{array}\right.\\ &= n\cdot u[n] \end{aligned}

Example 6: Convolution of causal real exponential γnu[n]\gamma^{n}\, u[n], γ≠1\gamma \neq 1, and unit step:

γnu[n]*u[n]={0n<0∑m=0nγmn≥0={0n<0γn+1−1γ−1n≥0=γn+1−1γ−1⋅u[n]\begin{aligned} \gamma^{n}u[n] * u[n] &= \left\{\begin{array}{cc} 0 & n < 0\\ \sum\limits_{m = 0}^{n} \gamma^{m} & n \geq 0 \end{array}\right.\\ &= \left\{\begin{array}{cc} 0 & n < 0\\ \frac{\gamma^{n+1} - 1}{\gamma - 1} & n \geq 0 \end{array}\right.\\ &= \frac{\gamma^{n+1} - 1}{\gamma - 1} \cdot u[n] \end{aligned}

Such tables of precomputed sums plus a set of properties allows us to preform convolution on a wide array of signals without integrating directly. Given arbitrary signals x1[n],x2[n]x_1[n], x_2[n], etc.

  • commutivity: x1[n]*x2[n]=x2[n]*x1[n]x_1[n] * x_2[n] = x_2[n] * x_1[n]

  • distributivity: x1[n]*(x2[n]+x3[n])=(x1[n]*x2[n])+[x1[n]*x3[n]]x_1[n] * (x_2[n] + x_3[n]) = (x_1[n]*x_2[n]) + [x_1[n]*x_3[n]]

  • associativity: x1[n]*(x2[n]*x3[n])=(x1[n]*x2[n])*x3[n]x_1[n] * (x_2[n] * x_3[n]) = (x_1[n]*x_2[n]) * x_3[n]

  • index shift: x1[n−m1)*x2[n−m2]=[x1*x2][n]|n→m1+m2x_1[n-m_1)*x_2[n-m_2] = \left. [x_1 * x_2][n] \right|_{n\rightarrow m_1 + m_2}

  • multiplicative scaling, a,b∈ℂa,b\in\mathbb{C}: (ax1[n])*(bx2[n])=ab(x1[n]*x2[n])(ax_1[n])*(bx_2[n]) = ab(x_1[n] * x_2[n])
